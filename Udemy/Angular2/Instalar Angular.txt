Instalar Angular

	https://victorroblesweb.es/2016/04/11/instalar-angular-2-hola-mundo/
	
	1. Instalar Node.js
	2. Crear un directorio para proyecto Angular
	3. Crear el fichero tsconfig.json
	4. Añadimos el fichero typings.json
	5. Creamos el fichero package.json
	6. Ejecutar el comando: npm install
	7. Creamos el directorio app y dentro un archivo llamado app.component.ts
	8. Creamos el fichero main.ts en el directorio app.
	9. Finalmente en la raíz del proyecto creamos el fichero index.html.
	10. Ejecutar el comando: npm start
